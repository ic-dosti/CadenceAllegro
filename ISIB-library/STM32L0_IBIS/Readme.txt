===============================
IBIS files for STM32L0 devices:
===============================

The package contains :


   + IBIS Files for STM32L0_11_21 devices on different packages :
           - lqfp32 
           - qfn20 
           - qfn28
           - qfn32 
           - tssop14
           - tssop20
           - wlcsp25

   + IBIS Files for STM32L0_31_41 devices on different packages :
           - lqfp32
           - lqfp48
           - qfn28
           - qfn32
           - tssop20
           - wlcsp25
           - qfn28 only for STM32L031GxUxS device

   + IBIS Files for STM32L0_51_52_53_62_63 devices on different packages :
           - lqfp32
           - lqfp48
           - lqfp64
           - tfbga64
           - ufqfn32
           - wlcsp36

   + IBIS Files for STM32L0_71_72_73_81_82_83 devices on different packages :
           - bga64
           - bga100
           - lqfp32
           - lqfp48
           - lqfp64
           - lqfp100
           - qfn32
           - wlcsp49

   + IBIS Files for STM32L0x0 Value Line devices on different packages :
           - tssop14
           - tssop20           
           - lqfp32 (16KB flash)
           - lqfp32 (64KB flash)
           - lqfp48
           - lqfp64 (64KB flash)
           - lqfp64 (128KB flash)
